## hm-javascript-session

Hello Mentor の課題で制作した、sessionStorage を使用したモーダル表示の実装です。

## 概要

ブログの記事ページにアクセスした際、5 秒後にモーダルを表示します。
一度表示されたモーダルは、セッションが切れるまで再表示されません。

## 実装内容

-   `<dialog>` 要素を使用したモーダル表示
-   Web Animations API を使ったフェードイン／フェードアウト
-   sessionStorage を使った表示状態の管理

## 仕様

-   ブログの記事ページにアクセスすると 5 秒後にモーダルを表示する
-   モーダルを一度表示すると、同一セッション中は再表示しない
-   セッションが切れた場合、再度アクセスすると 5 秒後にモーダルを表示する
-   モーダルはボタンまたは Esc キーで閉じることができる

## 使用しているストレージ

-   sessionStorage
-   キー名：`modalShown`
-   保存値：`true`（文字列）

## 使用目的

sessionStorage の特徴である
「ブラウザを閉じるまで状態を保持する」挙動を理解するための学習用実装です。
